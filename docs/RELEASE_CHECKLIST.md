# Obelisk Release Checklist

Use this checklist before tagging a release candidate.

## Build and Packaging

- [ ] `make clean && make` completes on a correctly configured host
- [ ] `build/kernel.elf` is produced
- [ ] `userland/build/init` and `userland/build/sysctl` are produced
- [ ] `build/rootfs.tar` contains `/sbin/init` and `/sbin/sysctl`
- [ ] `build/obelisk.iso` is generated by `grub-mkrescue`

## Boot Validation

- [ ] QEMU boot reaches kernel banner
- [ ] Filesystem, IPC, and sysctl init paths report successful startup
- [ ] No panic during idle loop runtime smoke test
- [ ] Debug and safe GRUB entries both boot

## ABI and Userland Sanity

- [ ] Syscall wrapper library links into userland binaries
- [ ] `sysctl` binary executes and prints usage/help
- [ ] `init` binary executes and prints startup banner
- [ ] New syscall changes are reflected in both kernel and userland

## Stability and Safety

- [ ] No known data-corruption regressions in AxiomFS test matrix
- [ ] No new kernel warnings introduced in touched subsystems
- [ ] Policy daemon config files are packaged into rootfs
- [ ] Recovery path for failed policy decisions is documented

## Documentation

- [ ] `README.md` build instructions match current tree
- [ ] `docs/ROADMAP.md` milestone status is current
- [ ] Known limitations and TODOs are documented for users
